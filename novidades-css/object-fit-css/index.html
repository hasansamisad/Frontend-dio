Propiedade Object-Fit

A propriedade object-fit define como o conteúdo de um elemento substituível (como img, video, iframe, embed, canvas) deve se ajustar às dimensões do seu contêiner.

Fill

Padrão. Redimensiona o conteúdo para preencher totalmente o contêiner, distorce se necessário.

object-fit: fill;

Contain

Redimensiona o conteúdo para caber completamente no contêiner, mantendo a proporção. Pode deixar espaço vazio (letterbox).

object-fit: contain;

Cover

Redimensiona o conteúdo para preencher completamente o contêiner, mantendo a proporção, mas pode cortar partes.

object-fit: cover;

None

Não redimensiona o conteúdo. O conteúdo mantém seu tamanho original, podendo transbordar o contêiner.

object-fit: none;

Scale-down

Redimensiona o conteúdo para o menor tamanho possível entre "none" e "contain". Se o conteúdo for maior que o contêiner, ele será redimensionado para caber, mas se for menor, permanecerá no tamanho original.

object-fit: scale-down;
