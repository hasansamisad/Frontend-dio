Propriedade Object-Position

A propriedade object-position define o alinhamento do conteúdo substituível (como <img>, <video>, etc.) dentro do seu contêiner, quando o object-fit não preenche completamente o espaço ou corta parte do conteúdo.

Valor padrão: center

Centraliza o conteúdo dentro do contêiner.

object-position: center;

Top Left

Alinha o conteúdo no canto superior esquerdo do contêiner.

object-position: left top;

Bottom Right

Alinha o conteúdo no canto inferior direito do contêiner.

object-position: right bottom;

Valores em porcentagem

Permite posicionar o conteúdo em qualquer ponto do contêiner usando porcentagens.

object-position: 80% 20%;

Valores em pixels

Também é possível usar valores em pixels para posicionar o conteúdo.

object-position: 30px 10px;
