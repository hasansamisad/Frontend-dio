Texto com imagem de fundo usando background-clip

Este é um exemplo de como usar o background-clip para aplicar gradientes ao texto. O uso de -webkit-background-clip: text permite que o fundo do texto seja preenchido com um gradiente, tornando-o visualmente interessante.

Certifique-se de que o navegador suporte a propriedade -webkit-background-clip para que o efeito seja visível. Este recurso é amplamente suportado em navegadores modernos, mas pode não funcionar em todos os navegadores.
