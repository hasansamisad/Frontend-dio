Atributo action na tag form vai falar para onde os dados fornecidos via usuário vão ser mandados, geralmente o endereço do servidor

Atributo method Post na tag form vai falar como os dados fornecidos via usuário vão ser enviados, geralmente via POST E o method get vai falar que os dados vão ser enviados via URL

Para senha o metodo get não é bom pois envias os dados para o servidor, ficando explicito a senha
